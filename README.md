# IP Address Management (IPAM) Tool

A Flask-based IPAM tool for subnet calculations, VLAN planning, wildcard mask generation, host range analysis, and CSV export.

## Live Demo:
https://ipam-tool.onrender.com

## Features

- Subnet Calculator
- Host Range Calculation
- Wildcard Mask Generation
- VLAN Planner
- CSV Export
- Input Validation

## Technologies

- Python
- Flask
- HTML
- CSS
